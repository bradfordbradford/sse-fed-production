    Hover & Active State

    To mark the current page, add a class of 'active' to the list item

    <!-- Hover/Active Example-->
<li class="active"><a href="#"><span class="ss-toolbox"></span>My Portfolio</a></li>

    This creates active states for both the inner navigation on the side and top nav dropdown menu.
