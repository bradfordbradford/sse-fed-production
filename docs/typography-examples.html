    There's Links. And then there's Links with SS-Icons…

    sse_typography.scss

    This is a plain old link. If a color is set on the body class, the hover state will match.


    <a href="#">a - This is a link.</a>

    Add eye-catching icons in a matter of keystrokes. This is a link with a SS-Icon. Isn't it iconic?

    <a class="ss-action" href="#">a - This is a link with an icon.</a>

    For a full list of available Symbolicons, check out the icon docs:

    • Block Symbolicons: webfonts > ss-symbolicons-docs.html
    • Social Symbolicons: webfonts > ss-social-docs.html
